What are databases?

A structured set of data

Databases are a collection of information; they are used for creating manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or non-relational database. Relationships mean a set of out data is related to another section of our data. Meanwhile, in a non-relational database, there is not a common related section of our data. Our databases are put into sections called schemas, inside each schema each data set is organized into tables, those tables contain columns and rows to house that particular set of data.

A stack of information!

Storage

All databases are just a bin of storage. This storage is housed on your harddrive. Often times larges companies will have floors upon floors of just database storage (think NSA styled). Our computer stores information on the harddrive, we are able to create new information, read information, update information, and delete information.

Searching

Since there could potentially be a lot of information a database stores, one problem we might run into is finding the data we are looking for. Imagine we had a room full of books and you were asked to find the Coding Dojo Algorithm Book. Well you might start picking up the book, checking the title, and moving on to the next until you find it. Thats very slow as most libraries discovered. Instead, if we were able to ID each item, we can find it quicker.
